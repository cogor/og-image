### DevTools

Nuxt OG Image uses [Nuxt DevTools](https://devtools.nuxt.com/) to provide a live preview of your OG Images.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  devtools: { enabled: true },
})
```
